This project is the  Web App Mobile Automation where User can automatically open the app, Fill up the Form with user information, Add to Cart product, Validate the Amount and Proceed.


<h2>üöÄ Project Features</h2>

"General Store APK Project" main functionalities Contains: 

*  User automatically open the app,
*  Input some requrement information form,
*  Add to cart the product,
*  Validate the amount,
*  Proceed the button,
*  Go to Google browser for Native App feature's.
